# hyprconf

My hyprland config.

This is designed specifically for Hyprland `0.55.0` and above, as since `0.55.0`, it has switched to using Lua for config.

This repo is primarily maintained on [Codeberg](https://codeberg.org/XDuskAshes/hyprconf). If you're already reading this there, good job! :D

This repo is also maintained on [Github](https://github.com/XDuskAshes/hyprconf) because Nix is really annoying in flakes with anything outside of Github, and when eventually the nixpkgs unstable repo ups the version there to `0.55.0` and I get my config where I want it, this repo will provide a wrapper package that contains my config.
